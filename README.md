# Web-Crawler-In-Python-With-Beautiful-Soup
This is a crawler that will scrape all the quotes of the website and store it into excel file

Use-Case:
A company has approaced you that is studying human behavior. The task is to give them a webcam, which can detect the motion or any movement in front of it. After detection it should return a csv file which will track the start & end time of the object.
Pre-Requisites:
* Editor: Any python Editor
* Language: Python 3.7 or greater
* Library: BeautifulSoup, SeleniumWebdriver
Run:
* scrape.py is the execution file
